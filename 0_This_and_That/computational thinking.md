## 실수 표현

부동소수점의 변환 과정

1. 십진수 -> 이진수 변환 (정수부 + 실수부)
2. 이진수 -> 부동소수점 표현(부호, 가수, 지수)
3. 부호 -> 0 for positive, 1 for negative
4. 지수 -> 127-biased
5. 가수 -> hidden bit 처리
